## Парсер веб страниц технических спецификаций автомобилей


### Поддерживаемые сайты

1. [Каталог auto.ru](https://auto.ru/catalog/cars/)


### Запуск

1. `scrapy crawl autoru_catalog -o autoru_catalog.csv -t csv`
